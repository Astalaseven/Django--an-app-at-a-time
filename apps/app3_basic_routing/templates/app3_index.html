Basic routing

Try to navigate to these urls :

  • ./prefix/: the statically prefixed URL.
  • .: this page (does nothing).
  • ./world/: a similar result than the previous apps.
  • ./You/: we can change the name displayed since it's extracted from the URL.
  • ./You/Hey/: we can add the optional parameter.

Here we use static links, but we can also create them dynamically. We'll see that in the next app.
