Liens

Même liens que dans l'app précédente, mais générés dynamiquement. Essayez ces liens :

  • {{ first_link }}/strong>
  • {% url 'hello' %}
  • {% url 'include' as third_link %} {{ third_link }}
  • {% url 'hello_name' 'world' %}
  • {% url hello_name 'You' as hello_link %} {{ hello_link }}
  • {% url hello_name prefix='Hey' name='You' as prefixed_link %} {{ prefixed_link }}

Générer les liens dynamiquement permet de changer facilement de vues, d'urls our même d'apps.
