Links

This is the same as the previous app, but we generate them dynamically. Try these links :

  • {{ first_link }}/strong>
  • {% url 'hello' %}
  • {% url 'include' as third_link %} {{ third_link }}
  • {% url 'hello_name' 'world' %} : a similar result than the previous apps.
  • {% url hello_name 'You' as hello_link %} {{ hello_link }}
  • {% url hello_name prefix='Hey' name='You' as prefixed_link %} {{ prefixed_link }}

Generating dynamically links ensure you can change views, urls or entire apps much more easily.
